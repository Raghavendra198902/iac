# CMDB Agent Downloads

This directory contains downloadable CMDB agents for various platforms.

## Available Agents:
- Windows Agent (x64, x86)
- Linux Agent (x86_64, aarch64)
- macOS Agent (Intel, Apple Silicon)
- Android Agent (ARM, ARM64)
- Cloud/Container Agent (Docker, Kubernetes)

## Checksums:
All downloads include SHA-256 checksums for verification.
